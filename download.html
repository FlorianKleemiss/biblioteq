Download

Desktop Version

    Please visit https://sourceforge.net/project/showfiles.php?group_id=129976 to download the latest version of BiblioteQ. Some dependencies are as follows: Qt 4.8.x or Qt 5.x, and YAZ 4.x or YAZ 5.x. PostgreSQL 8.x, PostgreSQL 9.x, and SQLite 3.x are supported database engines.

Online Version

    Please visit https://code.google.com/p/biblioteqonline for the online version of BiblioteQ. If the page is missing, please try https://github.com/search?utf8=%E2%9C%93&q=biblioteqonline.
